# Sober Friend

## Project Description
Sober Friend is a supportive application designed to help individuals in their journey toward sobriety. This project aims to provide a platform where users can track their progress, find resources, and connect with a community of like-minded individuals.

## Features
- Track sobriety days
- Journal entries to record thoughts and feelings
- Community forums for support and sharing experiences
- Resources for sobriety, including articles and hotlines

## How to Use
1. Clone the repository to your local machine.
2. Install the necessary dependencies using `npm install`.
3. Run the application with `npm start`.
4. Follow the instructions in the application to create an account or log in. 
5. Start tracking your sobriety journey today!
